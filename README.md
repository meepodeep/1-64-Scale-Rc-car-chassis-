# why does this exist 
Every week before finals for the last 2 years I have designed another revision of an RC car. This is this semester's version, inspired by a Hot Wheels Datsun 240Z I found at the grocery store is a 1/64 scale car chassis. 

# instructions 
# Reciver
1. Desolder the headers if needed; however, you may not have to, depending on what you are trying to use the chassis for
# Chassis
1. Resin 3d print all of the parts.
# Assembly
1. Glue in the motor
